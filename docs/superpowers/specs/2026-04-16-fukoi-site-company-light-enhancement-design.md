# Fukoi Site Company Light Enhancement Design

## Goal

在保留当前 `fukoi.co` 暗色、高级、游戏工作室气质与项目展示节奏的前提下，小幅增强法人主体信息，使首页与 `STUDIO` 页更适合作为对外正式公司网站使用。

## Direction

本次不重做首页结构，不把网站改成传统企业官网。整体策略是“品牌仍为主角，公司信息成为补强层”。

采用以下视觉与信息策略：

- 首页首屏仅增加一行极小的品牌归属说明，不改变当前 Hero 主标题和项目视觉主次。
- 复用现有首屏之后、项目展示之前的内容位置，将其调整为更克制的 `ABOUT FUKOI` 区块。
- 页脚补充法人信息，但维持暗色、低调、编辑式排版。
- `STUDIO` 页在保留现有视觉骨架的前提下，补成“品牌介绍 + 公司能力说明 + 当前推进事项”。

## Home

### Hero

在主标题上方增加一行极小辅助说明文字：

`FURI STUDIO / 株式会社富鯉`

处理方式：

- 字号明显小于 Hero 标题
- 作为品牌归属说明，不做高对比标签
- 颜色和间距保持克制，不抢主标题和项目视觉

### About Module

保留当前 Hero 后、项目区前的位置，调整为一个更简洁的 `ABOUT FUKOI` 区块。

内容严格使用以下日文正文，不改写：

`株式会社富鯉は、日本に拠点を置く独立法人として、ゲームプロダクトの開発・運営および海外市場展開に取り組んでおります。AI活用を中核としたゲーム開発体制を構築し、複数プロジェクトにおいて実運用を進めています。企画、アート制作、実装、ローカライズ、運営準備を含む一連のプロセスを通じて、良質なゲームコンテンツをより広い市場へ届けることを目指しています。`

正文下方补一行简洁公司信息：

`株式会社富鯉｜ゲーム開発・運営・海外市場展開｜東京都品川区東五反田一丁目8番1号 シルバープラザ五反田｜contact@fukoi.co`

版式要求：

- 不做传统企业卡片矩阵
- 使用窄栏正文、细线、低对比信息行
- 手机端保持合理行宽和段落节奏
